# 🎬 Movie App

![App Screenshot](./assets/screenshots/Home.png)

## 🌟 Features

- Browse popular movies
- Search for movies
- Add movies to watchlist
- View detailed information about a movie
- Category wise movie listing

## 🛠️ Tech Stack

- Flutter
- Firebase
- TMDB API
